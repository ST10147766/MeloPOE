Description
A comprehensive web application for the Gift of the Givers Foundation to manage disaster relief efforts, coordinate volunteers, and facilitate resource allocation during emergencies.

Usage Guide
Unit Testing: Validates the functionality of individual code components and core business rules in isolation.

Integration Testing: Ensures proper interaction and data flow between different system modules and database operations.

Performance Testing: Assesses application stability and responsiveness under high user loads and stressful conditions using JMeter.

User Experience Testing: Gathers feedback from real or simulated users to evaluate interface usability, workflow efficiency, and accessibility compliance.
For Administrators
1.	Access the admin dashboard at /Admin
2.	Manage disasters - Add, edit, and monitor disaster incidents
3.	View reports - Access donation and volunteer statistics
4.	Manage users - Oversee user accounts and permissions
For Volunteers
1.	Register - Create a volunteer account
2.	Browse tasks - View available relief tasks
3.	Update availability - Set your available time slots
4.	Track contributions - Monitor your volunteer hours
For Donors
1.	Make donations - Contribute goods or monetary donations
2.	Track donations - View donation history and status
3.	Anonymous giving - Option to donate anonymously
